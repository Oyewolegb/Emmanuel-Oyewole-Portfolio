Project 3: Countdown Timer

The Countdown Timer project is a simple web application that allows users to set a countdown by entering the hours, minutes, and seconds. Once the user clicks "Start," the timer begins counting down, updating every second until it reaches zero. When the countdown finishes, an alert notifies the user that the time is up. The user can also reset the timer at any time.
